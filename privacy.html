Política de privacidad

GainAI · Última actualización: junio 2026

Resumen: tus datos de entrenamiento nunca salen de tu dispositivo. La única excepción es cuando usás IA con tu propia API key (BYOK): en ese caso enviamos datos del ejercicio a Groq o Cohere para generar la sugerencia.

1. Datos que almacenamos localmente

GainAI guarda toda la información en el almacenamiento local de tu navegador (localStorage), bajo la clave gym_app_state_v2. Nosotros no tenemos servidores propios y nunca recibimos ni accedemos a tus datos.

Los datos almacenados incluyen:

  • Rutinas de entrenamiento (días, ejercicios y configuración)
  • Historial de sesiones (series, pesos, repeticiones y fechas)
  • Perfil personal opcional: peso corporal, altura, edad y sexo (usados solo para personalizar sugerencias de IA)
  • Preferencias de la app: tema, idioma, tiempo de descanso, sonido y vibración
  • API keys de Groq y Cohere que el usuario introduce (ver sección 2)

Todo permanece en tu dispositivo. Si borrás el caché del navegador o desinstalás la PWA, esos datos se eliminan permanentemente.

2. Uso de IA y envío de datos a terceros

GainAI ofrece sugerencias de carga inteligentes. La cadena de prioridad es:

  • Lógica local (sin internet, sin datos externos): siempre disponible.
  • Groq (groq.com): si el usuario provee su propia API key, al solicitar una recomendación se envía a Groq: nombre del ejercicio, historial reciente de series (peso y repeticiones) y, si está configurado, el perfil personal (edad, sexo, peso corporal). No se envía información de identificación personal.
  • Cohere (cohere.com): idéntico al flujo de Groq. Se usa como fallback si Groq falla o no hay key configurada.

Las API keys se guardan localmente en tu dispositivo y se envían directamente a Groq/Cohere como cabecera de autenticación en cada solicitud. GainAI nunca las almacena ni las transmite a ningún otro servidor.

Consultá la política de privacidad de cada proveedor para entender cómo procesan los datos que reciben:

  • Política de privacidad de Groq
  • Política de privacidad de Cohere

3. Analítica y tracking

GainAI no incluye analítica de terceros, cookies de seguimiento, píxeles de conversión ni ninguna otra tecnología de rastreo. No creamos perfiles de usuario ni vendemos datos.

4. Cuentas de usuario

GainAI no requiere ni ofrece cuentas de usuario. No hay registro, login ni servicio de autenticación de ningún tipo.

5. Cómo exportar o borrar tus datos

Exportar: en Ajustes → Datos → "Exportar JSON" descargás un archivo con todo tu historial.

Borrar: en Ajustes → Datos → "Reiniciar todo" eliminás permanentemente todos los datos locales. Podés también borrar el caché del sitio desde las opciones de tu navegador.

6. Menores de edad

GainAI no está dirigida a menores de 13 años. No recopilamos intencionalmente información de menores.

7. Cambios a esta política

Si realizamos cambios relevantes, actualizaremos la fecha en la parte superior de este documento. Como no tenemos forma de contactarte (no hay cuentas), te recomendamos revisar esta página ocasionalmente.

8. Contacto

Si tenés preguntas sobre esta política escribinos a: mzeacardenas@gmail.com

Privacy Policy

GainAI · Last updated: June 2026

Summary: your workout data never leaves your device. The only exception is when you use AI with your own API key (BYOK): in that case we send exercise data to Groq or Cohere to generate the suggestion.

1. Data stored locally

GainAI stores all information in your browser's local storage (localStorage), under the key gym_app_state_v2. We have no servers of our own and never receive or access your data.

Stored data includes:

  • Workout routines (days, exercises, and configuration)
  • Session history (sets, weights, reps, and dates)
  • Optional personal profile: body weight, height, age, and sex (used only to personalize AI suggestions)
  • App preferences: theme, language, rest timer, sound, and haptics
  • Groq and Cohere API keys entered by the user (see section 2)

Everything stays on your device. If you clear browser cache or uninstall the PWA, that data is permanently deleted.

2. AI usage and data sent to third parties

GainAI offers intelligent load suggestions. The priority chain is:

  • Local logic (no internet, no external data): always available.
  • Groq (groq.com): if the user provides their own API key, when requesting a recommendation the following is sent to Groq: exercise name, recent set history (weight and reps), and, if configured, the personal profile (age, sex, body weight). No personally identifying information is sent.
  • Cohere (cohere.com): identical flow to Groq. Used as a fallback if Groq fails or no key is configured.

API keys are stored locally on your device and sent directly to Groq/Cohere as an authentication header in each request. GainAI never stores them on or transmits them to any other server.

See each provider's privacy policy to understand how they process received data:

  • Groq Privacy Policy
  • Cohere Privacy Policy

3. Analytics and tracking

GainAI does not include third-party analytics, tracking cookies, conversion pixels, or any other tracking technology. We do not build user profiles or sell data.

4. User accounts

GainAI does not require or offer user accounts. There is no registration, login, or authentication service of any kind.

5. How to export or delete your data

Export: go to Settings → Data → "Export JSON" to download a file with your full history.

Delete: go to Settings → Data → "Reset all" to permanently erase all local data. You can also clear the site cache from your browser settings.

6. Children

GainAI is not directed at children under 13. We do not knowingly collect information from minors.

7. Changes to this policy

If we make significant changes, we will update the date at the top of this document. Since we have no way to contact you (no accounts), we recommend checking this page occasionally.

8. Contact

If you have questions about this policy, write to us at: mzeacardenas@gmail.com
